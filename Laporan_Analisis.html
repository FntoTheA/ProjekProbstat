Penanganan: Median Imputation

Mengapa median, bukan mean?

  • Robust terhadap outlier ekstrem (misal: PDRB > 1 miliar)
  • Lebih menjaga bentuk distribusi asli
  • Cocok untuk data non-normal
  • Proporsi missing kecil (~2% per variabel) → cukup efektif

Probabilitas dengan Distribusi Normal

mean_ipm <- mean(data_clean$ipm)
sd_ipm   <- sd(data_clean$ipm)

prob_ipm_80 <- 1 - pnorm(80, mean = mean_ipm, sd = sd_ipm)
ipm_p90     <- qnorm(0.90, mean = mean_ipm, sd = sd_ipm)

cat(sprintf("P(IPM > 80) berdasarkan distribusi normal = %.4f\n", prob_ipm_80))
## P(IPM > 80) berdasarkan distribusi normal = 0.1172
cat(sprintf("Nilai IPM pada persentil ke-90            = %.2f\n", ipm_p90))
## Nilai IPM pada persentil ke-90            = 80.78

Kesimpulan

Temuan Utama

Aspek Temuan
IPM Rata-rata ~sedang; ~32% wilayah IPM > 75
Kemiskinan ~49% wilayah di atas rata-rata nasional

Rekomendasi

  1. Wilayah IPM < 60 — alokasi sumber daya lebih besar untuk pendidikan & kesehatan
  2. Infrastruktur digital — prioritas di wilayah akses internet rendah
  3. Pengentasan kemiskinan — perluas bantuan sosial dan pemberdayaan ekonomi
  4. Ketenagakerjaan — program pelatihan kerja dan dukungan UMKM
  5. Kualitas data — perbaiki sistem pengumpulan untuk meminimalisir missing value
